# Gamified Campaign SaaS Research Notes (2026-02-19)

## Benchmarked Platforms
- Woobox (campaign builder: giveaways, instant win, forms, polls)
- ShortStack (templates incl. wheel, scratch, code redemption, white-label)
- Woorise (lead-gen forms + quizzes + giveaways)

## Common Product/UX Patterns
1. **Fast first interaction**: one clear CTA above the fold and minimal friction.
2. **Template-first setup**: visually distinct campaign styles with reusable presets.
3. **Progressive feedback**: progress bars, score increments, micro-celebrations.
4. **Reward anticipation**: delayed reveal + suspense animation before final result.
5. **Mobile first controls**: large touch targets, sticky actions, simplified nav.
6. **White-label polish**: per-brand themes (colors, typography, card radius, texture).
7. **Performance guardrails**: lightweight animation loops and instant route rendering.

## UI Trends Applied in this refactor
- Glassmorphism + depth layers (blurred cards over vivid gradient fields)
- Soft “orb” ambient backgrounds
- High-contrast pill navigation with icon labels
- Cute animated loading state
- Micro-interactions in games (burst, shuffle, urgent pulse)
- Reward panel emphasis with clearer hierarchy
